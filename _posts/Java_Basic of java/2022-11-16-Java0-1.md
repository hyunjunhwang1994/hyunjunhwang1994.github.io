문제점.
1. JDK 1.8 ( ARM ), 최신 이클립스 ( ARM ) 돌리면
이클립스는 2020 09 버전부터 JDK11 이상 버전만 지원하므로...
1. JDK 1.8 ( ARM ), 예전 이클립스 ( x64 ) 실행 시 에러 발생.
2. JDK 1.8 x64, 예전 이클립스 ( x64 ) 실행 시 작동하나 비정상적 로딩이라든지
거의 사용 불가.



결론적으로 이클립스 최신 ARM 버전을 JDK17로 일단 설치한 뒤
설정에서 JRE, 컴파일러를 1.8로 변경하여 사용하고 있습니다.



- 이클립스 설치

https://www.eclipse.org/downloads/



최신 버전의 AArch64로 받은 후, 실행시켜 줍니다.



일단 17.0.05로 되어 있는 상태로 설치폴더 놔두고 그대로 설치 및 LAUNCH(실행)까지 시켜 줍니다.
