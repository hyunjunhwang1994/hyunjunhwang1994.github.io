## 1. 별 찍기

피라미드 형식으로 1,3,5,7 개수만큼의 별 모양 찍기  
사실 알고리즘 입문(옛날에 많이 하던..)의 대표적인 문제인 별찍기를 해보았다.



```java

// 난이도 하


public class quiz2 {
    public void solution(int star) {



        for(int i=0; i<star; i++){

            for(int j=1; j<star-i; j++){
                System.out.print(" ");
            }
            for (int j=0; j<i*2+1; j++){
                System.out.print("*");
            }

            System.out.println();

        }

    }

    public static void main(String[] args) {
        quiz2 method = new quiz2();
        int star = 20;
        method.solution(star);
    }
}

```



## 2. 몫의 경우의 수 찾기.

N을 3과 5로 나누었을 때 몫이 적은 경우의 수 찾기   
3과 5로 나누어지지 않는다면 -1 리턴

ex) N = 21




3 x 7 = 21❌

<hr>
  
5 x 3 = 15  
3 x 2 = 6 

result = 5;🅾️



|n|result|
|---|---|
|21|5|



|n|result|
|---|---|
|4|-1|




```java
// 난이도 중하

public class quiz1 {

    public static void main(String[] args) {

        int N = 11;
        solution(N);


    }


    public static void solution(int N){

        int quotient = 0;
        int remainder = 0;

        int temp1 = 0;
        int temp2 = 0;
        int temp3 = 0;

        quotient = N / 5; // 몫
        remainder = N % 5; // 나머지

        // 5로 한 번에 나누어떨어진다면,
        // 몫이 제일 적을게 분명하므로 바로 리턴
        if(remainder == 0){
            System.out.println(N/5);
            return;
        }



        // ex) N = 57

        int i = 0;
        while (true){

            //1.      5 *   11  = 55
            //2.      5 *   10  = 50
            //3.      5 *   9  = 45
            temp2 = 5 * (quotient - i);


            // temp2 가 음수라면 5와 3으로 나누어떨어지지 않음 -1 리턴
            if(temp2 < 0){
                System.out.println("-1");
                    return;
            }


            //1.      57 - 55 = 2
            //2.      57 - 50 = 7
            //3.      57 - 45 = 12
            temp3 = N - temp2;


            //1.      2 % 3 == 2
            //2.      7 % 3 == 1
            //3.      12% 3 == 0
            if((temp3%3) != 0){
                i++;
            }else{

                //          12 / 3 = 4
                temp1 = temp3 / 3;

                //                 9   +   4          
                temp3 = (quotient - i) + temp1;
                System.out.println(temp3);// 13
                return;
            }


        }

    }
}


```



## 3. 상하좌우 큰 수 찾기

5x5 2차원 배열이 있을 때 해당하는 수가 상하좌우 배열의 값보다 크다면 별로 표시하기




```java
// 난이도 중 ~ 중상

public class quiz3 {
    public void solution(int[][] arr1) {


        // 상 -> 우 -> 하 -> 좌
        int[] dx = {-1, 0, 1, 0};
        int[] dy = {0, 1, 0, -1};

        int len = arr1.length;


        boolean sw = false;

        for(int i = 0; i<len; i++){
            for(int j = 0; j<len; j++){

                for(int k=0; k<4; k++){
                    int nx = i + dx[k];
                    int ny = j + dy[k];
                    

                    
                    if(nx >= 0 && nx < len && ny >= 0 && ny < len && arr1[nx][ny] >= arr1[i][j]){

                        System.out.print(arr1[i][j]);
                        sw = false;
                        break;

                    }
                    
                    
                    // 상하좌우의 값 모두(최소 2개, 최대 4개)보다 커야 하므로 위의 조건에 들어가지 않는다면
                    // 별 표시
                    sw = true;

                }
                if(sw == true){
                    System.out.print("*");
                }
            }

            System.out.println();
        }
    }

    public static void main(String[] args) {
        quiz3 method = new quiz3();
        int[][] arr1 = { {3,4,1,4,9}, {2,9,4,5,8}, {9,0,8,2,1}, {7,0,2,8,4}, {2,7,2,1,4} };
                           // 3 4 1 4 *
                           // 2 * 4 5 8
                           // * 0 * 2 1
                           // 7 0 2 * 4
                           // 2 * 2 1 4
        method.solution(arr1);
    }
}

```
