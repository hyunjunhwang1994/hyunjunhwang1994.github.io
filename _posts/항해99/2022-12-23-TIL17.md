오늘은 일단 피보나치 수 까지만.. 풀어 보려고 한다.

어쩌다가 됬다.. 피보나치 수를 구현하고 1234567의 나머지를 구하는 과정에서
오버플로우를 고치느라 시간이 좀 걸렸다.

```java

    public int solution(int n){
        int answer = 0;

        int n0 = 1;
        int n1 = 1;

        for(int i = 2; i<n; i++){

            answer = (n0 + n1) % 1234567;
            n0 = n1;
            n1 = answer;

        }


        return answer;
    }
```
